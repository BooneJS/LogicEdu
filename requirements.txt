# USAGE INSTRUCTIONS:
# 
# For End Users (just want to use the library):
#   pip install manimlogic
# 
# For Developers (want to contribute to the project):
#   1. Clone the repository
#   2. Create virtual environment: uv venv
#   3. Activate virtual environment: source venv/bin/activate (Linux/Mac) or venv\Scripts\activate (Windows)
#   4. Install in development mode: uv pip install -e .
#   5. Install development dependencies: uv pip install -r requirements-dev.txt
#   6. Run tests: uv pytest
#   7. Format code: uv black .
#   8. Check code quality: uv flake8
#   9. Type checking: uv mypy manimlogic/
# 
# For Contributors:
#   - Write tests for new features in tests/
#   - Ensure all tests pass: uv pytest
#   - Format code before committing: uv black .
#   - Check for issues: uv flake8 && uv mypy manimlogic/
#   - Update documentation if adding new features
